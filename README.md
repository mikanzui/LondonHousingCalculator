# London Housing Calculator

Realistic affordability calculator for London first-time buyers.

## What is this?

A free, mobile-first web tool that answers two questions London buyers actually have:

1. **"Can I afford this?"** — Enter salary, deposit, property price → get monthly cost, risk rating, stamp duty
2. **"Where can I buy?"** — See which London boroughs you can actually afford, ranked and colour-coded

## Features

- Mortgage affordability calculator with built-in UK tax estimator
- LISA deposit tracker with government bonus projection
- Area Affordability Finder (10–15 London boroughs, expandable)
- Property comparison (side-by-side)
- Stamp duty calculator with first-time buyer relief
- Mobile-first, no frameworks, free to host

## Tech Stack

- Vanilla HTML / CSS / JS
- Chart.js (CDN) for charts
- Static JSON for borough data (ONS / Land Registry)
- Hosted on Netlify or Vercel (free tier)
